# Питч продукта:

Сайт каталог для выбора разных кружков, хобби и секций . На сайте по категориям можно выбрать спорт , языки  итд. 
Например, в спорте , можно выбрать по категориям зимние виды спорта или летние виды. 
После выбора определённого вида спорта , выходит список тренеров  и мест. 
Также у каждого тренера есть рейтинг.  На сайте не только тренера , но ещё учителя языков и так далее.
